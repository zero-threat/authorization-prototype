# Прототип авторизации

## Запустить

1. Качаем/клонируем репозиторий 
2. Открываем консоль операционной системы и устанавливаем json-server глобально (npm i -g json-server)
3. Там же прописываем путь до скачанного репозитория (cd .../authorization-prototype) 
4. После пишем json-server --watch db.json
5. Открываем код редактор, в нем открываем консоль (терминал)
6. Пишем npm run serve
